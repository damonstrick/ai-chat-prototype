# Icons for chat input bar (from Figma)

Export these from **Figma** so the demo matches the design 1:1.

**File:** [AI-x-TQ](https://www.figma.com/design/ecoDrlufp9kmeaImxEiVDo/AI-x-TQ) → frame **487-24847** (the screen with the chat input at the bottom).

1. **Plus icon** (dark grey circular button with white +)
   - In Figma: select the **plus button** (left side of the input bar).
   - Right-click → **Export** → PNG, 2x.
   - Save as: `icon-plus.png` in this folder.

2. **Microphone icon**
   - Select the **microphone** icon (right of the input field).
   - Export as PNG 2x.
   - Save as: `icon-mic.png`.

3. **Audio waveform icon**
   - Select the **waveform** icon (right of the mic).
   - Export as PNG 2x.
   - Save as: `icon-waveform.png`.

If you export the **whole input bar** as one image, you can use that as a background instead; otherwise use the three PNGs above and place them in this `icons` folder.
